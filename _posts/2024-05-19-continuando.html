Comecei minha jornada como desenvolvedor em 2020 e, desde então, venho desenvolvendo diversas aplicações. Mais adiante, vou adicionar mais posts para contar com mais detalhes sobre minha caminhada até começar a usar o framework Ruby on Rails e desenvolver alguns projetos. Atualmente, desenvolvo aplicações usando Ruby on Rails e, durante esses 3 anos, fui me aperfeiçoando no uso desse framework. Você pode visitar meus repositórios no meu perfil do GitHub nicollinoxx. Durante o desenvolvimento com Ruby on Rails, também aprendi a usar Active Record, incluindo validações, migrações e associações para manipulação de banco de dados, além de SQLite e PostgreSQL para armazenamento de dados. No front-end, tenho adquirido experiência com frameworks de estilização como Tailwind CSS e Bootstrap, que me permitem criar interfaces de usuário modernas e responsivas. Além disso, estudei metodologias de desenvolvimento como Scrum e Extreme Programming (XP), bem como alguns conceitos de design e experiência de usuário.
